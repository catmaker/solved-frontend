🌟 Level 1-1: 타입 에러 찾기
다음 코드에서 타입 에러가 발생하는 부분을 찾고 수정해보세요.

```typescript
let userName: string = "김철수";
let userAge: number = "25";
let isActive: boolean = 1;
let score: number = null;
```

각 변수에 할당된 값의 타입이 선언된 타입과 일치하는지 확인해보세요
TypeScript는 타입이 정확히 맞아야 해요
